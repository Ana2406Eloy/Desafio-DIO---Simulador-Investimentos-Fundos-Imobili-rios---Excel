# Desafio-DIO---Simulador-Investimentos-Fundos-Imobili-rios---Excel
Este desafio consiste no desenvolvimento de uma planilha em Excel para simulação de investimentos em fundos imobiliários (FIIs), permitindo calcular aportes mensais, tempo de investimento, patrimônio acumulado e dividendos, auxiliando o usuário na tomada de decisões financeiras.
